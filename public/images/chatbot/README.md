# Icône du Chatbot

Placez votre image d'icône du chatbot dans ce dossier avec le nom suivant :

**Nom du fichier :** `chatbot-icon.png`

L'image sera utilisée comme icône pour le bouton du chatbot sur la page d'accueil.

**Spécifications recommandées :**
- Format : PNG (avec transparence de préférence)
- Taille : 28x28 pixels minimum (sera redimensionnée automatiquement)
- Fond : Transparent ou orange (selon votre design)
