# Images de Desserts Africains

## Images Requises

Pour compléter la catégorie desserts, vous devez ajouter 2 images de desserts africains :

### Images à télécharger :

1. **de5.jpg** - Image pour "Thiakry Traditionnel"
   - Dessert à base de mil, lait caillé et vanille
   - Format recommandé : JPG, résolution minimale 800x600px
   - Nom du fichier : `de5.jpg`

2. **de6.jpg** - Image pour "Bissap Glacé"
   - Boisson/dessert à base d'hibiscus (bissap)
   - Format recommandé : JPG, résolution minimale 800x600px
   - Nom du fichier : `de6.jpg`

## Où trouver des images de desserts africains ?

### Suggestions de sources :
- **Thiakry** : Recherchez "thiakry dessert africain" ou "thiakry mil lait"
- **Bissap** : Recherchez "bissap hibiscus" ou "jus d'hibiscus africain"

### Sites recommandés :
- Unsplash (https://unsplash.com) - Recherchez "african dessert" ou "thiakry"
- Pexels (https://www.pexels.com) - Recherchez "african food dessert"
- Pixabay (https://pixabay.com) - Recherchez "thiakry" ou "bissap"

## Instructions

1. Téléchargez les 2 images de desserts africains
2. Renommez-les en `de5.jpg` et `de6.jpg`
3. Placez-les dans ce dossier : `public/images/desserts/`

## Structure finale attendue

```
public/images/desserts/
├── de1.jpg (Tiramisu Maison)
├── de2.jpg (Mousse au Chocolat)
├── de3.jpg (Salade de Fruits Frais)
├── de4.jpg (Crêpes Caramel)
├── de5.jpg (Thiakry Traditionnel) ← À AJOUTER
└── de6.jpg (Bissap Glacé) ← À AJOUTER
```

## Notes

- Les images doivent être de bonne qualité pour un affichage optimal
- Format JPG recommandé pour réduire la taille des fichiers
- Assurez-vous que les images correspondent bien aux desserts décrits
